FONTLOG for the KeymanWeb-OSK font

This file provides detailed information on the KeymanWeb-OSK Font Software.
This information should be distributed along with the KeymanWeb-OSK fonts
and any derivative works.

Basic Font Information

KeymanWeb-OSK is a font used for interface elements of KeymanWeb and
Keyman's mobile software. (http://keyman.com/)

ChangeLog

24 October 2017 (SIL font development team) KeymanWeb-OSK version 1.001
